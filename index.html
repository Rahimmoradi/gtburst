gtburst

gtBurst is a python GUI (Graphical User Interface) which can be used to analyze Gamma-ray Bursts and Solar Flares observed by the instruments onboard the NASA Fermi satellite (formerly known as GLAST).

No need to download it separately

gtburst is now part of the official Fermi Science Tools package (http://fermi.gsfc.nasa.gov/ssc/data/analysis/software/). Just download and install the ST and you will be able to run gtburst by issuing "gtburst" on the command line.

Update to the latest version

The version of gtburst contained in the ST is most probably not the last one. While the ST have a release cycle of months, updates and new features for gtburst are issued much more frequently.

Once you are in the main GUI, select "update to the latest version" from the "Update" menu, and gtburst will download and install the updates. Then you will have to reset your session, and you are done.
